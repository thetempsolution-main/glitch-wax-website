Project Overview:
This project involves designing and building a basic business website for a company called Glitch Wax, which produces and sells board wax for various board sports such as skateboarding, snowboarding, and longboarding. The primary goal of the website is to provide a central hub that the client can link from their social media, allowing customers to learn about the product and make purchases. The website will function as a simple storefront with two products and will integrate external payment processing through Square rather than using a custom-built checkout system. The site will also serve as a branding tool to help establish legitimacy and increase conversions. This is a low-cost initial build intended to demonstrate value and open the door for future paid work. The project also serves as a portfolio piece and potential college-related work.

Completed Work:
A client agreement has been established where a basic website is built for free upfront, with a $50 payment if the client accepts the site, along with a right of first refusal agreement for future work. The project scope has been clearly defined to include a home page, a contact page with a form, a store page, and two product pages for the single stick and double pack products. The technical approach has been decided as a static website using HTML, CSS, and JavaScript, with no custom backend or cart system, and external payment processing handled through Square checkout links. The development environment has been set up by creating a project folder named glitch-wax-website and opening it in Visual Studio Code. The base file structure has been created, including index.html, contact.html, store.html, product-single.html, product-double.html, styles.css, and script.js. A basic homepage scaffold has been created with HTML boilerplate, and both the CSS and JavaScript files have been properly linked and tested. Live preview functionality has been set up using the Live Server extension in Visual Studio Code.

The homepage has now been fully built out as a complete first version. It includes a hero section, a “What is board wax?” section, a redesigned feature section with supporting images, a “Why Glitch Wax?” brand section, a featured products section, and a final call-to-action section. A consistent navigation bar and footer have been implemented. A structured image system has been created using an images folder, and stock images have been added and wired into all relevant sections of the homepage. The feature section has been upgraded from simple text cards to a mixed image and content layout for a more professional appearance. The CSS has been refactored to use variables for colors, spacing, and styling, making future visual changes easy and centralized. The homepage is now visually complete and functional as a temporary but presentable solution.
